Choose your favorite strange sea creature
-----------------------------------------
Site allows user to vote on favorite of two sea creatures, responds to user's choice by displaying total votes. On voting, selected image tilts. Keeps a tally of votes for each creature. Link provided to information about each creature. Attached page talks about importance of Marine Conservation.
